# Flipkart End to End Project

## Description
Automation testing project for Flipkart using Selenium, Cucumber and TestNG.

## Technologies Used
- Java
- Selenium WebDriver
- Cucumber BDD
- TestNG
- Maven
- Extent Reports

## How to Run
1. Clone the repository
2. Import as Maven project in Eclipse
3. Run `runner.java` as TestNG Test
